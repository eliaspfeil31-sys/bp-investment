# BP Investment Group Dashboard

## Deployment auf Vercel (via GitHub)

1. Lade diesen Ordner auf GitHub hoch
2. Geh auf vercel.com → "Add New Project"
3. Wähle dein GitHub Repository aus
4. Vercel erkennt Vite automatisch — einfach auf "Deploy" klicken
5. Fertig! Deine Seite ist live unter `bp-investments.vercel.app`

## Lokale Entwicklung

```bash
npm install
npm run dev
```
